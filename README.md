# WebCMD
A linux-style command prompt for the web. Built with vanilla JavaScript. [Try it online!](https://lebster.xyz/projects/webcmd)

## Features
 - File system
 - Command flags
 - Custom themes
 - Command piping
 - Dynamic files
 - Array results

***Images + explanations for features coming soon!***
